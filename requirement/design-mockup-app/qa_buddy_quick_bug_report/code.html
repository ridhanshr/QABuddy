[Mobile] Login Button Unresponsive on Check-out Screen

Description

Steps to Reproduce:

  1. Navigate to the checkout flow on mobile device (iOS/Android).
  2. Proceed to the final payment confirmation screen.
  3. Tap the "Login" button located at the bottom of the prompt.

Actual Result:

The button registers a tap state but no action is triggered. The user is stuck on the prompt.

Expected Result:

Tapping the login button should dismiss the prompt and navigate the user to the Authentication screen.
